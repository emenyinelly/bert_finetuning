# BERT Fine-Tuning for Sentiment Classification

**Course Checkpoint**: Fine-tune a pre-trained BERT model on a sentiment analysis task.

## Objective
Demonstrate the ability to load, fine-tune, evaluate, and test a BERT model using Hugging Face Transformers.

## Dataset Used
- **IMDb Movie Reviews** (Binary Sentiment: Positive / Negative)
- 4000 training samples + 1000 test samples (subset for faster training)

## Model
- **bert-base-uncased**
- Fine-tuned for 3 epochs
